# An-lisis-de-Eficiencia-Operativa-Alura-Store
Este proyecto surge de una necesidad estratégica  de la cadena *Alura Store*. El objetivo es identificar cuál de las cuatro tiendas de la cadena presenta el rendimiento más bajo o la menor eficiencia operativa para ser **vendida**, permitiendo así financiar un nuevo emprendimiento sin afectar críticamente el flujo de caja global.
